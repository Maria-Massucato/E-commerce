# E-commerce
<h4>Trabalho Prático de Desenvolvimento Front-End.</h4>
<br>
<b>Tema: Template Profissional de E-commerce de Cosméticos</b>
<br>
<br>
O Bootstrap foi usado em todas as páginas. 
<br>
Na página index.html temos navbar que possuem referências para as outras três páginas, carrossel com três imagens e suas respectivas descrições, cards dos produtos com refêrencia para acessar a página de visualização do produto, carrossel de feedbacks e um footer com newsletter.
<br>
<br>
Todas as páginas possuem o mesmo navbar com as referências para as páginas carrinho.html, catalago.html e duvidas.html, e o footer é usado o mesmo em todas as páginas.
<br>
<br>
Na pasta img não é usado todas as imagens que estão lá, uma das imagens é a palheta de cores que é usada como base para as cores selecionadas para a criação deste site.
<br>
<br>
Na pasta css, estão os arquivos de estilização das páginas. Os arquivos catalago.css e produto.css estão separados individualmente, pois eles não possuem padronização.
<br>
<br>
Na pasta js, o arquivo script.js possuem validação de formulário newsletter e carrinho. O arquivo filtros.js, como o nome mesmo diz é o filtro da página catalago.html.
<br>
<br>
Na página catalago.html temos os filtros de produtos na lateral do site e alguns cards com botton para adicionar ao carrinho, porém ele não é funcional.
<br>
<br>
Na página duvidas.html, ficam as principais dúvidas divididas em categorias.
<br>
<br>
Na página carrinho.html temos os produtos, ao todo quatro produtos, onde eles podem ser apagados e adicionarmos mais dos mesmos produtos existentes dentro deste carrinho.
<br>
<br>
Na página produto.html, fica informações detalhadas do produto, como: preço, descrição, quantidade, adicionar ao carrinho e efetuar compra.
<br>
<br>
O site por completo está responsivo.
<br>
